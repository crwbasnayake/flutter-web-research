# Sample Flutter Web Project

This is a sample project that is not well structured, but is used to test some capabilities of Flutter web. 

The project uses most of the sample codes given by the Flutter sample apps. You can use this project as a reference to understand how Flutter web works and its capabilities.

You can see the Firebase hosted sample website to get an idea of what the project looks like: https://atrad-rnd.firebaseapp.com/#/

Please note that this is just a sample project and not meant for production use. Use it at your own risk.

## Setup

To run this project, you will need to have Flutter installed on your machine. You can find the installation instructions for Flutter on the official website: https://flutter.dev/docs/get-started/install

Once you have Flutter installed, you can clone this repository and run the following command to run the project:
'flutter run -d chrome'

This will run the project in the Chrome browser.
